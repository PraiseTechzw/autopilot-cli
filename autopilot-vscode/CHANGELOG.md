# Changelog

## [0.1.0] - 2026-03-28

### Initial Release
- **Sidebar Integration**: Real-time tree view of Autopilot status, repository details, and activity.
- **Status Bar Integration**: Native VS Code status bar item with Quick Action menu.
- **Settings Dashboard**: Webview-based configuration of `.autopilotrc.json`.
- **Live Log Streaming**: Color-coded, auto-scrolling log viewer for real-time process monitoring.
- **Notification Bridge**: Translates CLI events (push, conflict, errors) into VS Code toast notifications.
- **CLI Runner**: Robust, non-interactive execution of Autopilot commands directly from the editor.
